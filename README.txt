Constantes.php contient les constantes nécessaires à la connexion
DbConnect.php permet la connexion à la db
DbOperations.php permet la manipulation de la db avec le CRUD
Client.php contient une classe client manipulable

Sur l'index.php, pour qu'une opération du CRUD fonctionne, il faut écrire l'opération nécessaire dans la zone de texte adéquate (écrire "create" pour créer un client par exemple).
Le journal contient les descriptions des manipulations horodatées.
